Linear regression cheat sheet on house prices¶

Quick reminder:

Objective of linear regression is to linearly approximate a target variabe Y (sale price of a home) using a set of p explanatory variables X1, X2, ..., Xp such as square meter, ....

In other words, we want to find coefficients \begin{equation} \beta_0, \beta_1, .. \beta_p \end{equation}

such as we could predict the price a home will be sold using some of its observed features as such:

\begin{equation} \hat{Y_i} = \beta_0 + \sum\limits_{j=1}^{p}\beta_j X_j \end{equation}

For the sake of simplicity, we will consider only one varibale $X_1$ to explain our price. We will find the best $(\hat{\beta_0}, \hat{\beta_1})$ that minimizes the loss on the training set and will further approximate the price of a test house i as such:

\begin{equation} \hat{Y_i} = \beta_0 + \beta_1 X_i \end{equation}

\begin{equation} Y_i = \hat{Y_i} + \epsilon_i = \beta_0 + \beta_1 X_i + \epsilon_i \end{equation}

Interesting checks¶

As $\epsilon$ is orthogonal to $ F = Vect(1, X_1,.., X_p)$, $\epsilon$ is orthogonal to $1 \in F$ :

\begin{equation} \langle\epsilon, \overrightarrow{1}\rangle = \epsilon.\overrightarrow{1} = \sum\limits_{i=1}^{n}{\epsilon_i} = 0 \end{equation}

y_train has been computed using the $\beta$ which is a computed value and thus differ very slightly from theorical one ==> We don't have pure 0 for our checks.

\begin{equation} \langle \epsilon, \hat{Y}\rangle = 0 \end{equation}

Also, always as $\epsilon$ is orthogonal to $ F = Vect(1, X_1,.., X_p)$, $Y = \hat{Y} + \epsilon = X.\hat{\beta} + \epsilon$ and $ X.\hat{\beta} \in F$ , we should have:

$${\lVert Y\rVert}^2 = {\lVert \hat{Y}\rVert}^2 + {\lVert \epsilon\rVert}^2 $$

The loss minimisation view¶

R Square¶

The coefficient of determination $R^2$ of the prediction is defined by: $$ R^2 = 1 - \frac{{\lVert Y-\hat{Y}\rVert}^2}{{\lVert Y-\bar{Y}\rVert}^2}$$

In [203]:
def square_norm(x):
    return (x * x).sum()

R2 = 1 - square_norm(y_test - y_hat_test) / square_norm(y_test - y_test.mean())

$R^2$ is equal to the square correlation between $Y$ and $\hat{Y}$¶

https://economictheoryblog.com/2014/11/05/proof/

$$corr(Y, \hat{Y}) =\frac{cov(Y, \hat{Y})}{{\lVert Y-\bar{Y}\rVert}{\lVert Y-\hat{Y}\rVert}} = \langle Y - \bar{Y}, \hat{Y} - \bar{Y}\rangle$$
